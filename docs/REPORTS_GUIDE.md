ASLM Reports Guide
==================

Overview
- Reports provide filtered listings and CSV exports across key modules: Assets, Audits, Checkouts, Leased Assets, Maintenance, Reservations, Status, Transactions, plus Contracts/POs/Software.

Access
- Navigate: /reports
- Permissions: reports.view (view), reports.export (CSV export). Admins have both.

Common Filters
- Date range, vendor, product, site, location, status, and per‑page size depending on the module.

Modules
- Assets: /reports/assets — filter by vendor, product, assigned/unassigned, search.
- Audits: /reports/audits — list audits and drill into variance.
- Checkout: /reports/checkout — list open/closed checkouts with due dates.
- Leased Assets: /reports/leased-assets — list assets with active leases and lease windows.
- Maintenance: /reports/maintenance — open/scheduled/completed maintenance.
- Reservations: /reports/reservations — pending/approved history.
- Status: /reports/status — asset counts by status.
- Transactions: /reports/transactions — operational events summary.
- Procurement: /reports/contracts, /reports/purchase-orders, /reports/software — vendor/product based reports.

Exporting
- Click Export in the toolbar (requires reports.export) to download CSV. The export respects current filters.

Tips
- Use the per‑page setting for quick pagination while previewing.
- For recurring exports, bookmark a filtered URL and use Export.
